# DevEnvCpp

Base Docker container that serves as my dev environment for C++ and possibly some other languages.

I am going to use this as a base to start from to teach some lessons on my YouTube channel at [youtube](https://www.youtube.com/@ChristopherPisz/)
That way everyone will have the same development environment and there will be less guessing about hardware/software someone else has when they ask questions.

## Important - This is the most basic branch

I have built on top of this concept and created other branches.

See the following branches:

### DevContainerCpp

Building upon the concept, we can wse VSCode's ability to automatically create, configure, and run one or more containers, images, and volumes to develop your project in, for you.
